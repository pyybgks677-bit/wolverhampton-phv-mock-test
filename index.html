About this mock test

This website provides an independent practice mock test for drivers preparing for the Wolverhampton PHV Knowledge Test. The questions are presented in a multiple-choice format so you can practise selecting answers and reviewing your performance.

You can use the mock test in two modes: a practice mode that allows review of incorrect questions, and an exam-style mode where results are shown at the end. The time limit shown is provided for practice.
