Guide

Practical revision tips and a simple approach to using this mock test effectively. This guide is general information only and is not official advice.

How to get the most from the mock test

  • Start in Practice mode so you can review incorrect answers.
  • After each attempt, review the questions you got wrong and retry them untimed.
  • Use the Flag feature to mark questions you want to revisit.
  • When you are consistent, switch to Exam mode for timed attempts.

Revision approach

A simple way to improve is to build accuracy first and then build speed:

  1. Accuracy phase: practise slowly and focus on understanding.
  2. Speed phase: practise timed attempts and keep answers consistent.
  3. Consistency phase: aim for repeated passes, not just one high score.

Keyboard shortcuts

  • A/B/C/D – select an answer
  • Left / Right arrows – previous / next question
  • F – flag / unflag
  • S – skip
